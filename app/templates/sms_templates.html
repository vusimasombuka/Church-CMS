📋 Available Variables (Copy & Use)

Variable Description Copy
{name} Person's first name (works in ALL templates)
{event_title} Event name (e.g., 'Youth Conference')
{event_date} Event date (e.g., '25 December 2026')
{days_until} Days until event (e.g., '90', '30')
{services} Service times (for visitor_followup only)
